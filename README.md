## üöÄ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/Arnav-Kumar1/portfolio.git
cd portfolio
```

2. **Install Frontend Dependencies**
```bash
cd frontend
yarn install
```

3. **Run the Application**
```bash
yarn start
```

The application will open at `http://localhost:3000`
